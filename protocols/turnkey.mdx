Turnkey is a wallet infrastructure platform that provides secure key management, wallet operations, and transaction signing across multiple blockchains. Turnkey offers comprehensive SDKs for various platforms and a robust REST API for wallet and key management.

## Overview

Turnkey provides developers with enterprise-grade wallet infrastructure:

- **Secure Key Management**: Hardware-backed private key storage and management
- **Multi-Chain Support**: Support for multiple blockchain networks
- **Wallet Operations**: Create, import, export, and manage wallets
- **Transaction Signing**: Sign transactions and raw payloads
- **Policy Management**: Create and manage policies for access control
- **User Management**: Create and manage users with authentication
- **OAuth Integration**: OAuth 2.0 authentication support
- **Smart Contract Interfaces**: Create and manage smart contract interfaces

**Package**: `@turnkey/react`

**Package**: `@turnkey/react-native`
