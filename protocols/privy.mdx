Privy is a wallet infrastructure platform that enables developers to create wallets, manage users, and handle transactions across multiple blockchains. Privy provides SDKs for client-side and server-side integration, along with a comprehensive REST API.

## Overview

Privy provides developers with comprehensive wallet infrastructure:

- **Multi-Chain Support**: Ethereum, Solana, Spark BTC, and other chains
- **Wallet Management**: Create, import, export, and manage wallets
- **User Management**: Create users with linked accounts and wallets
- **Transaction Handling**: Sign messages and send transactions
- **Policy Controls**: Create policies to control wallet spending and contract interaction
- **Account Funding**: Native onramp and offramp capabilities
- **KYC Integration**: Built-in KYC verification support

## SDKs & Clients

### Client-side SDKs

#### React

**Package**: `@privy-io/react-auth`

**Installation**:
```bash
npm install @privy-io/react-auth
```

**Usage**:
```jsx
import { PrivyProvider } from '@privy-io/react-auth';

function App() {
  return (
    <PrivyProvider appId="your-app-id">
      <YourApp />
    </PrivyProvider>
  );
}
```

#### React Native

**Package**: `@privy-io/react-native`

**Installation**:
```bash
npm install @privy-io/react-native
```

#### Swift

**Package**: Privy iOS SDK

**Installation**: Available via Swift Package Manager

#### Android

**Package**: Privy Android SDK

**Installation**: Available via Maven/Gradle

#### Flutter

**Package**: Privy Flutter SDK

**Installation**: Available via pub.dev

#### Unity

**Package**: Privy Unity SDK

**Installation**: Available via Unity Package Manager

### Server-side SDKs

#### Node.js

**Package**: `@privy-io/server-auth`

**Installation**:
```bash
npm install @privy-io/server-auth
```

**Usage**:
```javascript
import { PrivyClient } from '@privy-io/server-auth';

const client = new PrivyClient({
  appId: 'your-app-id',
  appSecret: 'your-app-secret'
});
```

#### Python

**Package**: `privy-python`

**Installation**:
```bash
pip install privy-python
```

#### Java

**Package**: Privy Java SDK

**Installation**: Available via Maven

#### Rust

**Package**: `privy-rs`

**Installation**:
```bash
cargo add privy-rs
```

## APIs

### Wallets

<Expandable title="Wallet APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Create Wallet | POST | Create a new wallet |
| Get Wallet | GET | Retrieve wallet details by ID |
| Update Wallet | PATCH | Update wallet settings |
| Get Wallets | GET | List all wallets |
| Get Balance | GET | Get wallet balance |
| Get Transactions | GET | Get wallet transaction history |
| Export Wallet | POST | Export wallet |
| Import Wallet | POST | Import an existing wallet |

</Expandable>
