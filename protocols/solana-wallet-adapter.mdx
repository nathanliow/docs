Solana Wallet Adapter is a standard interface library for connecting wallets in Solana applications. It provides a unified API for wallet connections, making it easy to support multiple wallets in your application.

## Overview

Solana Wallet Adapter provides developers with a standardized wallet connection interface:

- **Multi-Wallet Support**: Connect to Phantom, Solflare, Glow, and other Solana wallets
- **Unified API**: Single interface for all wallet operations
- **React Integration**: React hooks and components for easy integration
- **Type Safety**: Full TypeScript support
- **Transaction Signing**: Sign and send transactions through connected wallets
- **Event Handling**: Listen to wallet connection and disconnection events

## SDKs & Clients

### React

**Package**: `@solana/wallet-adapter-react`

**Installation**:
```bash
npm install @solana/wallet-adapter-react @solana/wallet-adapter-react-ui @solana/wallet-adapter-wallets
```

**Usage**:
```jsx
import { WalletProvider, useWallet } from '@solana/wallet-adapter-react';
import { WalletModalProvider, WalletMultiButton } from '@solana/wallet-adapter-react-ui';
import { PhantomWalletAdapter, SolflareWalletAdapter } from '@solana/wallet-adapter-wallets';

const wallets = [
  new PhantomWalletAdapter(),
  new SolflareWalletAdapter(),
];

function App() {
  return (
    <WalletProvider wallets={wallets} autoConnect>
      <WalletModalProvider>
        <WalletMultiButton />
        <YourApp />
      </WalletModalProvider>
    </WalletProvider>
  );
}
```

### Base Library

**Package**: `@solana/wallet-adapter-base`

**Installation**:
```bash
npm install @solana/wallet-adapter-base
```

**Usage**:
```typescript
import { WalletAdapter } from '@solana/wallet-adapter-base';

// Use wallet adapters to connect and interact with wallets
const adapter = new PhantomWalletAdapter();
await adapter.connect();
const publicKey = adapter.publicKey;
```

### Wallet Adapters

Individual wallet adapters are available for specific wallets:

- `@solana/wallet-adapter-phantom` - Phantom wallet
- `@solana/wallet-adapter-solflare` - Solflare wallet
- `@solana/wallet-adapter-glow` - Glow wallet
- `@solana/wallet-adapter-backpack` - Backpack wallet
- And many more wallet-specific adapters
