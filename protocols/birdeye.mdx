Birdeye is a comprehensive real-time token price and analytics platform for Solana and other blockchains. It provides accurate prices, live trading activity, trending tokens, and liquidity insights through powerful APIs.

## Overview

Birdeye aggregates data across multiple decentralized exchanges (DEXs) to provide:
- **Real-time token prices**: Ultra-fast price updates across all Solana tokens
- **Market analytics**: Market cap, liquidity, trading volume, and trends
- **Wallet analytics**: Portfolio tracking, PnL calculations, and transaction history
- **Multi-chain support**: Solana, Ethereum, Sui, BNB Smart Chain, and more
- **Comprehensive APIs**: Easy integration for price feeds, analytics, and wallet data

## SDKs & Clients

### JavaScript/TypeScript

**Package**: `@birdeye/birdeye-api` or `birdeye-api`

```bash
npm install birdeye-api
```

**Usage**:
```javascript
import { BirdeyeAPI } from 'birdeye-api';

const api = new BirdeyeAPI('YOUR_API_KEY');
const price = await api.getTokenPrice('token_address');
```

### Python

**Package**: `birdeye-api`

```bash
pip install birdeye-api
```

**Usage**:
```python
from birdeye import BirdeyeAPI

api = BirdeyeAPI('YOUR_API_KEY')
price = api.get_token_price('token_address')
```

### REST API

Direct REST API access with any HTTP client:
- Base URL: `https://public-api.birdeye.so`
- Authentication: API key in headers
- Documentation: Available in API docs
