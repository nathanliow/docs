Jupiter is the leading DEX aggregator on Solana, providing the best swap rates by routing trades across multiple DEXs. They offer comprehensive APIs for integrating swaps, price quotes, and DeFi functionality into your applications.

## Overview

Jupiter aggregates liquidity from all major Solana DEXs to provide:
- **Best swap rates**: Automatically finds the best prices across all DEXs
- **Smart routing**: Splits trades across multiple DEXs for optimal execution
- **Comprehensive API**: Easy integration for swaps, quotes, and more
- **Real-time data**: Live price feeds and market data

## SDKs & Clients

### JavaScript/TypeScript

**Package**: `@jup-ag/api` or `@jup-ag/core`

**Installation**:
```bash
npm install @jup-ag/api
```

**Usage**:
```javascript
import { QuoteResponse, SwapResponse } from '@jup-ag/api';

// Get quote
const quote = await fetch(
  `https://quote-api.jup.ag/v6/quote?inputMint=${inputMint}&outputMint=${outputMint}&amount=${amount}`
).then(res => res.json());

// Execute swap
const swap = await fetch('https://quote-api.jup.ag/v6/swap', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ quoteResponse: quote, userPublicKey: publicKey })
}).then(res => res.json());
```

### Python

**Package**: `jupiter-python-sdk` or direct REST API calls

**Installation**:
```bash
pip install requests
```

**Usage**:
```python
import requests

# Get quote
quote = requests.get(
    'https://quote-api.jup.ag/v6/quote',
    params={
        'inputMint': input_mint,
        'outputMint': output_mint,
        'amount': amount
    }
).json()

# Execute swap
swap = requests.post(
    'https://quote-api.jup.ag/v6/swap',
    json={'quoteResponse': quote, 'userPublicKey': public_key}
).json()
```

### React Component

**Package**: `@jup-ag/react-hook` or `@jup-ag/react-ui`

**Installation**:
```bash
npm install @jup-ag/react-hook

  // Use quote data
}
```
