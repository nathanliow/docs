Helius is a comprehensive Solana infrastructure provider offering RPC endpoints, enhanced APIs, WebSocket services, and dedicated node solutions.

## Overview

Helius provides developers with powerful infrastructure tools to build on Solana. Their platform includes:

- **Enhanced RPC Endpoints**: High-performance RPC access with advanced features
- **Enhanced APIs**: Pre-indexed data and transaction parsing
- **WebSocket Services**: Real-time data streaming via Laser Stream
- **Dedicated Nodes**: Custom node infrastructure for enterprise needs

## Services

### Dedicated Nodes

Helius offers dedicated node infrastructure for enterprise applications:

**Features:**
- Single-tenant node infrastructure
- Custom configuration options
- High availability and reliability
- Priority support
- Custom RPC endpoints

**Specifications:**
- Standard tier: Single dedicated node
- Premium tier: High-performance nodes with enhanced specs
- Enterprise tier: Custom configurations and multi-node setups


## SDKs & Clients

### JavaScript/TypeScript

**Package**: `@helius-labs/helius-sdk` or use `@solana/web3.js` with Helius RPC endpoint

**Installation**:
```bash
npm install @solana/web3.js
```

**Usage**:
```javascript
import { Connection } from '@solana/web3.js';

const connection = new Connection('https://rpc.helius.dev/?api-key=YOUR_API_KEY');
// Use standard Solana RPC methods
```

For enhanced APIs, use direct HTTP requests or community SDKs.

### Python

**Package**: `solana-py` with Helius RPC endpoint

**Installation**:
```bash
pip install solana
```

**Usage**:
```python
from solana.rpc.api import Client

client = Client('https://rpc.helius.dev/?api-key=YOUR_API_KEY')
# Use standard Solana RPC methods
```

### REST API

Direct REST API access for enhanced APIs:
- Base URL: `https://api.helius.dev`
- Authentication: API key in query parameter or headers
- Documentation: Available in API docs

### WebSocket Client

For Laser Stream WebSocket connections:
- WebSocket URL: `wss://atlas-mainnet.helius-rpc.com`
- Authentication: API key in connection parameters
- Supports standard WebSocket libraries in any language
