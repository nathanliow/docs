QuickNode is a blockchain infrastructure provider offering high-performance RPC endpoints, enhanced APIs, and developer tools for Solana and other blockchains. QuickNode provides reliable, scalable infrastructure for building Web3 applications.

## Overview

QuickNode provides developers with powerful infrastructure tools:

- **High-Performance RPC**: Fast and reliable RPC endpoints
- **Enhanced APIs**: Additional APIs beyond standard RPC methods
- **Multi-chain Support**: Solana, Ethereum, Polygon, BSC, and more
- **Analytics Dashboard**: Monitor usage and performance
- **Dedicated Nodes**: Custom node infrastructure for enterprise needs

## SDKs & Clients

### JavaScript/TypeScript

**Package**: Use `@solana/web3.js` with QuickNode RPC endpoint

**Installation**:
```bash
npm install @solana/web3.js
```

**Usage**:
```javascript
import { Connection } from '@solana/web3.js';

const connection = new Connection('https://your-endpoint.solana-mainnet.quiknode.pro/YOUR_API_KEY');
// Use standard Solana RPC methods
```

### Python

**Package**: `solana-py` with QuickNode RPC endpoint

**Installation**:
```bash
pip install solana
```

**Usage**:
```python
from solana.rpc.api import Client

client = Client('https://your-endpoint.solana-mainnet.quiknode.pro/YOUR_API_KEY')
# Use standard Solana RPC methods
```

### REST API

Direct REST API access:
- Base URL: Your QuickNode endpoint URL
- Authentication: API key in URL or headers
- Documentation: Available in QuickNode dashboard

### WebSocket Client

For WebSocket connections:
- WebSocket URL: `wss://your-endpoint.solana-mainnet.quiknode.pro/YOUR_API_KEY`
- Supports standard WebSocket libraries in any language
