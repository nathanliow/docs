## Stablecoin APIs

### Sphere

#### Transfers

<Expandable title="Transfer APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Create Transfer | POST | Create a new transfer between fiat and virtual currencies |
| Get Transfer by ID | GET | Retrieve transfer details by ID |
| List Transfers | GET | Get list of all transfers |

</Expandable>

#### Virtual Accounts

<Expandable title="Virtual Account APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Create Virtual Account | POST | Create a virtual account for on-ramp |
| Get Virtual Account | GET | Retrieve virtual account details by ID |
| Update Virtual Account | PATCH | Update virtual account settings |
| List Virtual Accounts | GET | Get list of all virtual accounts |
| List Transfers of Virtual Account | GET | Get transfers for a specific virtual account |
| Reactivate Virtual Account | PATCH | Reactivate a deactivated virtual account |
| Deactivate Virtual Account | PATCH | Deactivate a virtual account |

</Expandable>

#### CCTP OffRamp

<Expandable title="CCTP OffRamp APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Create CCTP OffRamp | POST | Create a CCTP off-ramp transfer |

</Expandable>

#### Transfer Fees

<Expandable title="Transfer Fee APIs">

| API Name | Type | Description |
|----------|------|-------------|
