## Balance APIs

### Helius

#### Token API
- **Get Token Balances**: Get token balances for addresses


### Birdeye

#### Wallet Portfolio API
- **Get Wallet Portfolio**: `GET /wallet/portfolio` - Token holdings and portfolio breakdown


### Moralis

#### Token API
- **Get Token Balances**: Get token balances for addresses


### QuickNode

#### Token API
- **Get Token Balances**: Get token balances for addresses
- **Get Token Accounts**: Get token account information


### Phantom

#### Get Tokens
- **Get Token Balance**: Get balance for a specific token
- **Get Wallet Tokens**: Retrieve all tokens held by a connected wallet
