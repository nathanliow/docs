## Event Metadata

APIs for retrieving event and market metadata, including event details, market information, series, tags, and search functionality.

### DFlow

#### Events API Endpoints

<Expandable title="Events APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Get Event | GET | Retrieve event details |
| Get Events | GET | Get list of events |
| Get Event Forecast Percentile History | GET | Get forecast percentile history for an event |
| Get Event Forecast Percentile History by Mint | GET | Get forecast percentile history by token mint |
| Get Event Candlesticks | GET | Get candlestick data for an event |

</Expandable>

#### Markets API Endpoints

<Expandable title="Markets APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Get Market | GET | Retrieve market details |
| Get Market by Mint | GET | Get market information by token mint |
| Get Markets | GET | Get list of markets |
| Get Markets Batch | POST | Get multiple markets in one request |
| Get Outcome Mints | GET | Get outcome token mints for a market |
| Filter Outcome Mints | POST | Filter outcome mints by criteria |
| Get Market Candlesticks | GET | Get candlestick data for a market |
| Get Market Candlesticks by Mint | GET | Get market candlesticks by token mint |

</Expandable>

## Streaming

Real-time data streaming via WebSocket connections for live updates on prices, trades, and orderbooks.

### DFlow WebSockets

<Expandable title="WebSocket Channels">

DFlow provides real-time data streaming via WebSocket connections for prediction markets:

| Channel Name | Description |
|--------------|-------------|
| Prices Channel | Real-time price updates for markets and tokens |
| Trades Channel | Stream trade events as they occur |
| Orderbook Channel | Real-time orderbook updates |

</Expandable>
