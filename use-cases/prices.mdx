## Current Token Prices

APIs for retrieving current token prices for SOL and SPL tokens.

### Birdeye

<Expandable title="Token Price API">

| API Name | Type | Path | Description |
|----------|------|------|-------------|
| Get Token Price | GET | `/token/price` | Get current token price and market data |
| Get Multiple Token Prices | GET | `/token/multiple-price` | Get prices for multiple tokens |

</Expandable>

### Jupiter

<Expandable title="Price v3 API">

| API Name | Type | Description |
|----------|------|-------------|
| price | GET | Get current token prices |

</Expandable>

### Moralis

<Expandable title="Price API">

| API Name | Type | Description |
|----------|------|-------------|
| Get Token Prices | GET | Get current token prices |
| Get Multiple Token Prices | GET | Get prices for multiple tokens in one request |

</Expandable>
