## Authentication

APIs and SDKs for connecting wallets, authenticating users, and managing wallet connections.

### Solana Wallet Adapter

#### Wallet Connection

<Expandable title="Wallet Connection APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Connect | Method | Connect to a wallet |
| Disconnect | Method | Disconnect from wallet |
| Auto Connect | Option | Automatically connect on page load |
| Select Wallet | Method | Select a specific wallet adapter |

</Expandable>

**Usage**:
```typescript
import { useWallet } from '@solana/wallet-adapter-react';

function MyComponent() {
  const { connect, disconnect, connected, publicKey } = useWallet();
  
  const handleConnect = async () => {
    await connect();
  };
  
  return (
    <button onClick={connected ? disconnect : handleConnect}>
      {connected ? 'Disconnect' : 'Connect Wallet'}
    </button>
  );
}
```

## Signing

APIs and SDKs for signing transactions, messages, and raw payloads.

### Solana Wallet Adapter

#### Transaction Signing

<Expandable title="Signing APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Sign Transaction | Method | Sign a transaction |
| Sign All Transactions | Method | Sign multiple transactions |
| Sign Message | Method | Sign a message |

</Expandable>

**Usage**:
```typescript
import { useWallet } from '@solana/wallet-adapter-react';
import { Transaction } from '@solana/web3.js';

function MyComponent() {
  const { signTransaction, publicKey } = useWallet();
  
  const handleSign = async () => {
    const transaction = new Transaction();
    // Add instructions to transaction
    const signed = await signTransaction(transaction);
    // Send signed transaction
  };
  
  return <button onClick={handleSign}>Sign Transaction</button>;
}
```

### Privy

#### Transaction Signing

<Expandable title="Transaction Signing APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Sign Message | POST | Sign a message using personal_sign |
| Send Transaction | POST | Send a transaction using eth_sendTransaction |

</Expandable>

**SDK Usage (React)**:
```jsx
import { usePrivy } from '@privy-io/react-auth';
import { useWallets } from '@privy-io/react-auth';

function MyComponent() {
  const { ready, authenticated } = usePrivy();
  const { wallets } = useWallets();
  
  const handleSignMessage = async () => {
    const wallet = wallets[0];
    if (!wallet) return;
    
    // Sign a message
    const signature = await wallet.signMessage({
      message: 'Hello from Privy!',
      encoding: 'utf-8'
    });
    
    console.log('Signature:', signature);
  };
  
  const handleSendTransaction = async () => {
    const wallet = wallets[0];
    if (!wallet) return;
    
    // Send a transaction
    const txHash = await wallet.sendTransaction({
      to: '0x...',
      value: '1000000000000000',
      chainType: 'ethereum'
    });
    
    console.log('Transaction hash:', txHash);
  };
  
  return (
    <div>
      <button onClick={handleSignMessage}>Sign Message</button>
      <button onClick={handleSendTransaction}>Send Transaction</button>
    </div>
  );
}
```

**SDK Usage (Server-side - Node.js)**:
```javascript
import { PrivyClient } from '@privy-io/server-auth';

const client = new PrivyClient({
  appId: 'your-app-id',
  appSecret: 'your-app-secret'
});

// Sign a message
const signature = await client.wallets.signMessage({
  walletId: 'wallet-id',
  message: 'Hello from Privy!',
  encoding: 'utf-8'
});

// Send a transaction
const txHash = await client.wallets.sendTransaction({
  walletId: 'wallet-id',
  method: 'eth_sendTransaction',
  params: {
    transaction: {
      to: '0x...',
      value: '1000000000000000'
    }
  },
  chainType: 'ethereum'
});
```

### Turnkey

#### Transaction Signing

<Expandable title="Transaction Signing APIs">

| API Name | Type | Description |
|----------|------|-------------|
| Sign Transaction | POST | Sign a transaction |
| Sign Raw Payload | POST | Sign a raw payload |
| Sign Raw Payloads | POST | Sign multiple raw payloads |
| Submit Transaction Intent | POST | Submit a transaction intent for broadcasting |
| Submit Raw Transaction | POST | Submit a raw transaction for broadcasting |

</Expandable>

**SDK Usage (TypeScript/JavaScript)**:
```typescript
import { TurnkeyClient } from '@turnkey/server';

const client = new TurnkeyClient({
  apiKey: 'your-api-key',
  apiSecret: 'your-api-secret'
});

// Sign a transaction
const signedTx = await client.signTransaction({
  walletId: 'wallet-id',
  transaction: {
    // Transaction data
  }
});

// Sign a raw payload
const signature = await client.signRawPayload({
  walletId: 'wallet-id',
  payload: Buffer.from('message to sign'),
  encoding: 'hex'
});

// Submit a transaction
const txHash = await client.submitTransaction({
  walletId: 'wallet-id',
  signedTransaction: signedTx
});
```

**SDK Usage (Python)**:
```python
from turnkey import TurnkeyClient

client = TurnkeyClient(
    api_key='your-api-key',
    api_secret='your-api-secret'
)

# Sign a transaction
signed_tx = client.sign_transaction(
    wallet_id='wallet-id',
    transaction={
        # Transaction data
    }
)

# Sign a raw payload
signature = client.sign_raw_payload(
    wallet_id='wallet-id',
    payload=b'message to sign',
    encoding='hex'
)

# Submit a transaction
tx_hash = client.submit_transaction(
    wallet_id='wallet-id',
    signed_transaction=signed_tx
)
```

**SDK Usage (React)**:
```jsx
import { useTurnkey } from '@turnkey/react';

function MyComponent() {
  const { signTransaction, signRawPayload } = useTurnkey();
  
  const handleSignTransaction = async () => {
    const signedTx = await signTransaction({
      walletId: 'wallet-id',
      transaction: {
        // Transaction data
      }
    });
    
    console.log('Signed transaction:', signedTx);
  };
  
  const handleSignPayload = async () => {
    const signature = await signRawPayload({
      walletId: 'wallet-id',
      payload: 'message to sign',
      encoding: 'hex'
    });
    
    console.log('Signature:', signature);
  };
  
  return (
    <div>
      <button onClick={handleSignTransaction}>Sign Transaction</button>
      <button onClick={handleSignPayload}>Sign Payload</button>
    </div>
  );
}
```
